> [!Note] 23种设计模式：
>
> - 创建型5种： 工厂方法、抽象工厂、原型、单例、构建器
> - 结构型7种： 适（适配）、桥（桥接）、组（组合）、装（装饰）、外（外观）、享（享元）、代（代理）—【代桥组装适享外】
> - 行为型11种：其他



**创建型** 创建型模式主要用于创建对象，为设计类实例化新对象提供指南。
